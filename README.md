# 7000LBSBAI - Programming for Business Analytics

## Module Repository

Welcome to the GitHub repository for the **Programming for Business Analytics** module on the MSc Business Analytics and AI programme at Liverpool John Moores University.

This repository contains teaching materials, workshop notebooks, datasets, and supporting resources used throughout the module.

## Module Overview

This module introduces students to programming for business analytics using Python. No previous programming experience is required.

The module focuses on developing practical analytical skills that are widely used in modern organisations, including:

* Data manipulation and cleaning
* Exploratory data analysis
* Data visualisation
* Statistical analysis
* Business problem solving
* AI-assisted analytics workflows

The emphasis is on applying programming techniques to real business problems rather than learning programming theory.

## Software Used

The module uses cloud-based tools to minimise software installation requirements.

### Google Colab

Workshop activities and assessments are completed using Google Colab.

Google Colab provides a browser-based Python environment that allows students to write and run code without installing specialist software.

Access Google Colab at:

https://colab.research.google.com

### DataCamp

DataCamp is used to support independent learning and skills development throughout the module.

### Canvas

Canvas is the primary source of module information, announcements, assessment details and lecture materials.

## Using Datasets in Google Colab

Most workshop datasets can be loaded directly from GitHub into Python using pandas.

Example:

import pandas as pd

url = "https://raw.githubusercontent.com/YOUR-REPOSITORY-PATH/datasets/retail_sales.csv"

df = pd.read_csv(url)

df.head()

*Workshop notebooks will provide the appropriate dataset links when required.*

## Working Practices

Students are encouraged to:

* Save copies of all notebooks regularly.
* Maintain backups in OneDrive.
* Keep assessment work organised and clearly labelled.
* Document the use of generative AI tools where appropriate.
* Develop reproducible analytical workflows.

## Generative AI

This module recognises that generative AI tools are increasingly used in business analytics practice.

Students may use approved AI tools to support learning and coding activities where permitted by module guidance. Refer to module Canvas page for full details of what is and is not recognised as approved use of AI.
Students remain responsible for checking, understanding and validating all outputs produced by AI systems.
